Simple Hello World application using qt5 for embedded Linux device.

Published at April 1, 2020 ·  2 min read

This post presents a simple example of how to create a qt5 application for a Linux embedded device that runs Wayland or x-server. I’m using Yocto build system. I already have a BSP for IMX8 + toolchain so. I just have to install qt5 on it. In the time that I wrote this page, I worked with sumo branch and had to do some work around over three bugs that I found during the compilation of qt5....
